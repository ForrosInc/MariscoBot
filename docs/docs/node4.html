Configuration

Youtube



should always be 1 except for debugging.
should contain the Youtube Search API key.

Language



should be set to a constant representing the array index that your language uses. By default, this is 0 for Spanish and 1 for English.

Language per Guild



By default, this is disabled. If used, it should be configured such that is the ID provided by and is set to whatever language should be used with that guild instead of the default.

Discord bot API



should be set to your bot's API key.
